## Installation

1. Use git to clone this repository.
   ```bash
   pacman -Sy git glibc
   git clone https://github.com/KateL2002/Easy-2-Install-Arch-Linux.git
   ```

2. Change work directory and run this script.
   ```bash
   cd Easy-2-Install-Arch-Linux
   bash setup.sh
   ```

3. Setup the installation.
